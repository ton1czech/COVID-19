# COVID-19 analýza

## 🌍 O Aplikaci

Webová aplikace bude sloužit pro souhrnný přehled dat o viru COVID-19, také bude zajišťovat informace o průběhu této nemoci. Bude možné si prohlédnout data všech zemí.
